# Figma Design Sample Project

A minimal project setup for implementing Figma designs using the Figma Desktop MCP server.

## Setup

The Figma Desktop MCP server is already configured and running at `http://127.0.0.1:3845/mcp`.

## Project Structure

```
figma-design-sample/
├── index.html       # Main HTML file
├── styles.css       # CSS styles
├── script.js        # JavaScript for interactivity
└── README.md        # This file
```

## Usage

1. Open your Figma design in Figma Desktop
2. Use Claude Code to fetch design information from Figma
3. Implement the designs in `index.html`, `styles.css`, and `script.js`
4. Open `index.html` in your browser to preview

## Development

Simply open `index.html` in your browser to view the page. No build process required!
